backdrop-vagrant
======================

A simple vagrant VM for backdrop built on CentOS 6.5 64 bit.

This should give you the latest checkout of backdrop and spin it up in a VM.

Installation
------------

`git clone https://github.com/btopro/backdrop-vagrant.git --recursive`
`cd backdrop-vagrant`
`vagrant up`
